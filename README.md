# Spring MVC 课程管理示例

基于 Spring Boot + Spring MVC 的简易课程管理系统，使用 JSP 提供交互界面，内存存储课程信息，并包含简单登录功能。

## 运行

确保安装了 JDK 17 与 Maven。

```bash
mvn spring-boot:run
```

应用启动后访问：

- 登录页: http://localhost:8080/login （默认账号：admin / 123456）
- 登录成功后自动跳转至首页 `/home`

## 功能概览
- 登录/退出：固定账号验证，登录后跳转首页。
- 首页：提供新增课程、查看课程列表入口。
- 新增课程：填写课程名、描述、授课日期（格式 `yyyy-MM-dd`，后台存毫秒）、可选人数、教室（下拉）、可选专业（多选）、授课方式（线上/线下）。
- 课程列表：展示所有课程名，点击查看详情。
- 课程详情：展示全部信息（含授课时间对应毫秒值），可进入编辑页面。
- 编辑课程：回填已有信息，可再次提交更新。

课程、教室和专业信息全部存储在内存中，适合本地演示与验证 Spring MVC 流程。
